Design System v2.1.0

Ntizar
Design System

Sistema de diseño completo con Liquid Glass UI. Azul y naranja. Un solo archivo CSS, sin dependencias, listo para cualquier proyecto. Esta página es la documentación completa.

Empezar a usar Referencia rápida
🤖 Para IAs: Este CSS se llama ntizar.css. Contiene 30+ clases reutilizables. Pega <link rel="stylesheet" href="ntizar.css"> y usa las clases documentadas abajo. Light-first, con modo oscuro automático y manual. Todos los valores son variables CSS sobreescribibles.
Paso 0

Instalación y Setup

Copia ntizar.css a tu proyecto y enlázalo. No hay dependencias. No hay build system. Opcionalmente incluye el filtro SVG para refracción real en Chrome.

1. Enlazar el CSS
HTML 
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="ntizar.css">
</head>
<body>
  <!-- Tu contenido aquí -->
</body>
</html>
2. (Opcional) Filtro SVG para Liquid Glass

Para activar la refracción real tipo Apple en Chrome/Chromium, pega este SVG dentro del <body> y añade la clase .glass-refract a los elementos glass. En otros navegadores, el fallback visual (blur + saturate) se aplica automáticamente.

HTML 
<!-- Pegar antes de cerrar </body> -->
<svg class="ntizar-svg-filters" xmlns="http://www.w3.org/2000/svg">
  <defs>
    <filter id="ntizar-glass-filter" x="-30%" y="-30%"
            width="160%" height="160%"
            color-interpolation-filters="sRGB">
      <feImage id="ntizar-displacement-img" href=""
        x="0" y="0" width="1" height="1"
        result="displacement_map" preserveAspectRatio="none"/>
      <feDisplacementMap in="SourceGraphic" in2="displacement_map"
        scale="28" xChannelSelector="R" yChannelSelector="G"
        result="refracted"/>
      <feGaussianBlur stdDeviation="1" in="refracted"
        result="blurred_for_spec"/>
      <feComposite in="refracted" in2="blurred_for_spec"
        operator="over"/>
    </filter>
  </defs>
</svg>

3. Core

Liquid Glass System

Tres niveles de intensidad (subtle, standard, intense), variantes de color (blue, orange), y specular highlight animado. En Chrome se aplica refracción óptica real via SVG filter.

🤖 Para IAs — Cuándo usar cada nivel:
• .glass-subtle → Fondos de sección, dividers, áreas de bajo contraste
• .glass-standard → Cards, paneles, drawers (nivel por defecto)
• .glass-intense → Modales, heroes, elementos prominentes
• Añadir .glass-specular para el highlight animado en hover
• Añadir .glass-blue o .glass-orange para tinte de color
• Añadir .glass-refract para refracción óptica (Chrome)
.glass-subtle

Nivel Subtle

Blur 8px, saturate 1.4. Para secciones, dividers, áreas de bajo contraste.

.glass-standard

Nivel Standard

Blur 16px, saturate 1.6. Cards, paneles, drawers. El nivel por defecto.

.glass-intense

Nivel Intense

Blur 28px, saturate 1.8. Modales, heroes, elementos prominentes.

.glass-blue

Tinte Azul

Añadir junto a glass-standard. Para elementos primarios o estados activos.

.glass-orange

Tinte Naranja

Añadir junto a glass-standard. Para alertas, CTAs secundarios, acento.

.card-gradient-border

Borde Gradiente

Borde azul→naranja con CSS mask. Sin SVG externo, sin JS.

Specular highlight

Añade .glass-specular para un rim-light superior y efecto shimmer en hover. Variantes: .specular-blue y .specular-orange.

18. Theming

Light & Dark Mode

Light mode por defecto. Dark mode se activa automáticamente con prefers-color-scheme: dark, o manualmente con las clases .theme-light / .theme-dark en el <html>.

🤖 Para IAs:
• El CSS es light-first. Si no haces nada, es claro.
• Si el OS del usuario está en dark mode, @media prefers-color-scheme: dark aplica automáticamente.
• Para forzar tema con JS: document.documentElement.classList.add('theme-light')
• Todas las variables se sobreescriben en dark. No necesitas cambiar clases de componentes.
JS 
// Toggle de tema
let dark = false;
function toggleTheme() {
  dark = !dark;
  document.documentElement.classList.toggle('theme-light', !dark);
  document.documentElement.classList.toggle('theme-dark', dark);
}

// Forzar light
document.documentElement.classList.add('theme-light');

// Forzar dark
document.documentElement.classList.add('theme-dark');
