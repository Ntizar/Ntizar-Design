Constellation

Packs modulares — carga sólo lo que uses

Cada pack es independiente y vive bajo el mismo namespace .nz. Sin colisiones, sin orden estricto, sin JavaScript obligatorio.

core
ntizar.css

Tokens, layout, componentes base, tipografía, glass, light/dark, focus accesible.

themes
ntizar.themes.css

5 skins (aurora, sunset, midnight, ocean, citrus) + paleta de charts coherente.

data
ntizar.data.css

KPIs, stat tiles, progress, meter, skeletons, avatars, timeline, tags.

charts
ntizar.charts.css

Contenedores para Chart.js / Apex / D3, sparklines y donuts CSS-only, paletas.

maps
ntizar.maps.css

Frame para Leaflet/Mapbox/MapLibre, overlays HUD, pins, popups, leyenda.

viz
ntizar.viz.css

Stages para three.js/canvas, fondos aurora animados, orbs, glow rings, scanlines.

motion
ntizar.motion.css

Reveal, rise, scale, glow-pulse, aurora-pan, shimmer, marquee, typing, hover-lift.

¿Qué incluye Constellation?
Core (tokens, base, objects, components, utilities) + 8 packs opcionales: themes, data, charts, maps, viz, motion, forms, ui, patterns.
¿Funciona con Tailwind / Bootstrap?
Sí. Todo está aislado bajo .nz. Convive sin colisionar.
¿Cómo cambio la marca?
Sobreescribe los tokens --nz-color-brand, --nz-color-accent y los --nz-gradient-* en tu propio CSS posterior.
¿Hay roadmap?
Sí, en SYSTEM.md. Próximos packs: editor, print, email.
